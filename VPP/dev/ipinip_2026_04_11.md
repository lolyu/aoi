# VPP IPinIP Tunnel — 2026-04-11


## valid packet forwarding
```
from scapy.all import *

# With ICMP inner
pkt = (Ether(dst="22:59:d0:d5:19:a9", src="3e:38:be:ea:b0:1c") /
       IP(src="10.20.20.10", dst="10.1.0.32") /
       IP(src="10.10.10.10", dst="20.20.20.20") /
       ICMP())

# Or with UDP inner
pkt = (Ether(dst="22:59:d0:d5:19:a9", src="3e:38:be:ea:b0:1c") /
       IP(src="10.10.10.10", dst="192.168.0.1") /
       IP(src="10.10.10.10", dst="20.20.20.20") /
       UDP(sport=1234, dport=5678) /
       Raw(b"D" * 64))

sendp(pkt, iface="eth28", count=10000)
```

vpp# create ipip tunnel src 192.168.0.1 p2mp outer-table-id 0
ipip0
