# syncd

```
/usr/bin/syncd --diag -u -s -B null -p /etc/sai.d/sai.profile -b /tmp/break_before_make_objects
```

* `--diag`: diagnostic shell
* `-u`: tempview
* `s`: synchronous mode
* `-B`: batch-polling counters
* `-p`: profile map file
* `-b`: break config
